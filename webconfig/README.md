A web application that allows you to configure Blue2Joy directly from any browser supporting the Web Bluetooth standard.

It provides an easy way to configure the mapping of HID controls to joystick port digital and analog pins.

Application communicates with Blue2Joy over Bluetooth, even while it is connected to HID devices (e.g., gamepads, mice).
